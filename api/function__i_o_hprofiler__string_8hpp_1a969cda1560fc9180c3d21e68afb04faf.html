std::vector<int> get_int_vector_parse_string(std::string input, const int _min, const int _max)¶

To get a vector of integer values with a range ‘n-m’.

Return a vector of integer value. The supported formats are: ‘-m’, [_min, m] ‘n-‘, [n, _max] ‘n-m’, [n, m] ‘n-x-y-z-m’, [n,m] \para input string \para _int int \para _max int

Return

std::vector<int>
